# Automation Exercise using Playwright for Web and API Testing
---

This is a Playwright testing framework designed to demonstrate playwright web and api testing example

### Pre-requisites
* Download and install Node.js
* Download and install any Text Editor like VS Code

### Setup Scripts
* Clone the repository into a folder
* Go to Project root directory and install Dependency: `npm install`
* All the dependencies from package.json would be installed in node_modules folder.

### Update Visual Code Settings
* Go to Visual Code Preference > Setting and search `formatOnSave` and enable/ON it.

## How to Run Test Locally

## Web Tests
* Go to the Project root directory and run command: npm run uiTest

## API Tests
* Go to the Project root directory and run command: npm run apiTest

## How to view default Playwright HTML report
* Go to the Project root directory and run command: npm run play-report

### CI/CD pipeline
* GitHub Actions to be triggered When a push or pull request are triggered
